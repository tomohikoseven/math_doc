# 数学の主張 (Math Doc)

数学の理論・定理や、ソフトウェア・IT関連の知識などを整理して公開しているドキュメントサイトのリポジトリです。
実際のサイトは以下のリンクからご覧いただけます。

👉 **[数学の主張 (https://mathdoc.ifdef.jp/)](https://mathdoc.ifdef.jp/)**

## 📚 主なコンテンツ

このサイト（リポジトリ）では、主に以下のテーマを扱っています。

- **数学系:** 
  - 実数論
  - 微分積分 (McGraw-Hill Calculus など)
  - 線形代数 (川久保線形代数 など)
  - 高校数学
  - 数学日記
- **ソフトウェア系:**
  - 各種ソフトウェア、ツール、ライブラリの備忘録や紹介
  - その他、IT関連の記事
- **その他:**
  - 数学以外の雑記やWebサイトについて

## 🛠 技術スタック

高速で美しいドキュメントサイトを構築するため、主に以下の技術を使用しています。

- **フレームワーク:** [Astro](https://astro.build/)
- **テーマ (ドキュメント):** [Starlight (by Astro)](https://starlight.astro.build/)
- **数式・PDF組版連携:** [Typst](https://typst.app/) (`rehype-typst`, `astro-typst`)
- **図解・ダイアグラム:** [D2](https://d2lang.com/) (`astro-d2`)
