　数学を勉強していて，思うことが多くある．例えば，数学の参考書・インターネット記事・YouTube動画で
解説される数学的な概念に自分の理解とズレがあり，少し苛ついてしまう．また定義・定理の数学的な
記述が示され，その具体例が提示されるだけで，それらの意味が解説されないので，作成者は本当にわかっているのか
疑問に思うこともある．数学は集合が基礎にあるので，集合的に考えたらどう関係するのか示してほしいと思う．

　このようなイラつきやもやっとしたことを解消する手段として，Webならある程度お金がかからずに表現できるかもしれないと考えた．
Webで表現しておくと，同じように考えた読者に届き，同じように解消できるかもしれない．そんなことから「数学の主張」は始まった．

## 忍者ホームページに落ち着くまでの経緯．
　Webサイト構築にあたって必要な条件があった．
1. 複雑な数式も表示できること．
1. 広告を載せて，収益化できること．
1. 低予算で構築できること．

　ブログといえばWordPressである．数学系ブログはいたるところにあり，数式の問題はないだろう．
広告はGoogleアドセンスで申請すれば通るだろうし，あとは低予算でWordPress構築できるところがあればよかった．
探した結果，[ロリポップ！レンタルサーバー](https://lolipop.jp/)に決まった．ライトプラン(約500円／月額，毎月契約更新)で開始した．

　3ヶ月ほどWordPressを運用していたときに問題が発生した．Googleアドセンスの申請が通らないのだ．
打つ手はすべてやったが，それでも申請が通らない．原因がわからず，1ヶ月過ぎる．

　また，数学系ブログは動的なWebページである必要がなく，ユーザへのページ表示速度が改善できるのではと気づいた．
WordPressはHTTPリクエストに対し，テンプレートや記事の中身を使ってWebページをクライアントに返却する．数式においてはクライアント側で描画することになる．
，動的なWebページ生成および数式のクライアント側描画で，ページの表示速度が遅くなることに

## 忍者ホームページにWebサイトを構築した背景

**メリット**

- Googleにインデックス登録できる．
  - [X Server Static](https://static.xserver.ne.jp/)上でWebサイトを構築してもインデックス登録できるが，その後エラーとなる．クローラーがWebページを見れなくなるようだ．有料版にすれば解決できそうではある．

**デメリット**

- Googleアドセンスに登録できない．
