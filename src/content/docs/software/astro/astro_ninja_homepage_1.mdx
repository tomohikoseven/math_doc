## 1. 事前準備：環境を整える

構築を始める前に、以下の2点を準備する。

### Node.js のインストール (Windows)

Astroを動かすには **Node.js (v18.14.1 以上)** が必須である。

<Steps>

1.  **公式サイトへアクセス**
    [Node.js 公式サイト](https://nodejs.org/ja/)にアクセスする。

2.  **LTS版をダウンロード**
    「LTS」と書かれたボタン（推奨版）をクリックして、インストーラー（.msiファイル）をダウンロードする。

3.  **インストーラーを実行**
    ダウンロードしたファイルを開き、基本的にすべて「Next」で進めてインストールを完了させる。
    - 「Tools for Native Modules」のチェックボックスが表示された場合は、チェックを入れたまま進めることを推奨する。

4.  **インストールを確認**
    ターミナル（PowerShellやコマンドプロンプト）を開き、以下のコマンドを入力してバージョンが表示されれば成功である。
    ```bash
    node -v
    npm -v
    ```

</Steps>

<Aside type="caution" title="注意">
  コマンドを打ってもバージョンが表示されない場合は、ターミナルを一度閉じてから開き直すか、Windowsを再起動すること。
</Aside>

### 忍者ホームページのアカウント作成

最終的な公開先として[忍者ホームページ](https://www.ninja.co.jp/hp/)のアカウントが必要となる。無料で登録できるため、今のうちに済ませておく。
- サーバー作成時、好きなドメイン名（URL）を決めることができる。
- 詳細は、公式サイトの案内に従って進めること。

---

## 2. Astroプロジェクトの初期化

Webサイトの本体を作成する。ターミナルを開き、プロジェクトを作成したいディレクトリに移動してから以下のコマンドを実行する。

<Steps>

1.  **プロジェクト作成コマンドを実行**
    ```bash
    npm create astro@latest
    ```

2.  **対話形式の質問に回答する**
    コマンドを実行するといくつか質問される。基本的には以下のように選択すればよい。
    - **Where should we create your new project?** -> `my-math-site` (任意のフォルダ名)
    - **How would you like to start your new project?** -> `Use blog template` または `Include sample files` (Starlightを後で入れる場合はどれでも可)
    - **Install dependencies?** -> `Yes`
    - **Initialize a new git repository?** -> `Yes`
    - **Do you plan to write TypeScript?** -> `Yes` (推奨)

3.  **Starlightの導入** （**推奨**）
    本サイトのような**ドキュメント形式**にする場合、Starlightを追加するのが最も近道である。
    ```bash
    npx astro add starlight
    ```

</Steps>

<Aside type="tip" title="テンプレートの利用">
  すでに構築済みのテンプレート（本サイトの構成に近いもの）を使いたい場合は、Astro公式のテンプレートギャラリーから選ぶことも可能である。
</Aside>

---

## 3. プロジェクトの構造を確認

作成されたフォルダ（`my-math-site`）の中身を確認する。主要なファイル構成は以下のようになる。

<FileTree>
- src/
  - content/
    - docs/
      - **index.mdx** (トップページ)
      - guide.md (記事ファイル)
  - assets/ (画像など)
- astro.config.mjs (設定ファイル)
- package.json (プロジェクト情報)
</FileTree>

---

## 4. ローカルサーバーでブラウザで確認する

ローカル環境でブラウザ確認を行う。以下のコマンドを入力する。

<Steps>

1.  **ディレクトリに移動**
    ```bash
    cd my-math-site
    ```

2.  **開発サーバーを起動**
    ```bash
    npm run dev
    ```

3.  **ブラウザでアクセス**
    ターミナルに表示されたURL（通常は `http://localhost:4321/`）をブラウザで開く。

</Steps>

「**Welcome to Starlight**」のような画面が表示されれば成功である。

## FAQ：Webサイト構築（ローカル編）のよくある質問

### Q: Node.jsのバージョンが古いとどうなるか？
**A:** Astroのインストール中にエラーが発生したり、開発サーバーが起動しなくなったりする。必ず **v18.14.1 以上** （**推奨はLTS版の最新**） を使用すること。

### Q: 「npm create astro」でエラーが出る
**A:** ネットワーク環境や、実行しているフォルダの書き込み権限を確認する。また、`node -v` でNode.jsが正しくインストールされているか再確認すること。

### Q: Starlight以外のテンプレートでも忍者ホームページに公開できるか？
**A:** 可能である。ただし、本ガイドでは数学ドキュメントに最適な **Starlight** を前提として解説を進める。
