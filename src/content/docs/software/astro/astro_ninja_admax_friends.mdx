Webサイトに広告を掲載して収益化を図ることに加え、さらに効率よく報酬を得る手段として「<b>忍者AdMaxFriends</b>」という仕組みが存在する。本記事では、この紹介プログラムの仕組みと、サイト運営者にとってのメリットを解説する。

## 1. 忍者AdMaxFriendsのしくみとは？

忍者AdMaxFriendsは、あなたが他者に『忍者AdMax』を紹介し、その方が新規登録して収益を上げることで、あなたにも紹介報酬が支払われる仕組みである。

<Card title="紹介プログラムの特長" icon="star">
  <FlatList>
  - <b>5%の継続報酬</b>：紹介された方が獲得した報酬の5%分が、毎月あなたにも入る（紹介された方の報酬が減るわけではない）。
  - <b>永続的なシステム</b>：単発の報酬ではなく、紹介された方が稼ぎ続ける限り、2ヶ月目以降も継続して報酬が発生する。
  - <b>人数の上限なし</b>：紹介する人数に上限はないため、新規登録者が増えるほど毎月の報酬額が雪だるま式に増える可能性がある。
  </FlatList>
</Card>

## 2. 忍者AdMaxの基本情報（紹介するメリット）

他者にサービスを紹介するにあたり、まずは忍者AdMax自体の魅力を再確認しておくことが重要である。これらのメリットを伝えることで、より多くの登録を促すことができる。

<FlatList>
- <b>審査なしで即スタート可能</b>：ブログやサイトを持っていれば、タグを貼るだけですぐに広告配信を開始できる。
- <b>表示報酬型とクリック報酬型</b>：広告がクリックされなくても、表示されるだけでPVに応じたポイントが貯まるシステムがある。
- <b>自動最適化</b>：訪問者や広告枠に合わせて、最も収益期待値が高い広告がオークション形式で自動選別・表示される。
</FlatList>

## 3. ポイントの交換手順

紹介報酬や通常の広告収益によって貯まったポイントの取り扱いルールである。

<Steps>

1.  <b>忍者ツールズポイントの獲得</b>

    獲得したポイントはすべて「忍者ツールズポイント」として付与され、専用ページで合計を確認できる。

2.  <b>500ptから交換可能</b>

    ポイントが500pt（500円相当）以上貯まると、「現金」または「PeXポイント」へ交換申請が可能となる。

3.  <b>有効期限に注意</b>

    ポイントには有効期限が設定されているため、一定期間内に新たなポイント獲得やログインを行うなど、失効を防ぐための管理が必要である。

</Steps>
