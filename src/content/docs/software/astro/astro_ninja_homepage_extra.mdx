Webサイトを公開したら、次なる目標は「**収益化**」である。本ガイドの構成（無料サブドメイン）ではGoogleアドセンスの審査通過が難しいため、審査なしで即導入できる「**忍者AdMax**」を活用しよう。

## 1. 忍者AdMaxが「最強の代替案」である理由

なぜ、Astro × 忍者ホームページの構成において忍者AdMaxが選ばれるのか、その魅力をまとめた。

<Card title="忍者AdMaxの3大メリット" icon="star">
  - **審査なしで即開始**: 登録したその日から広告を掲載し、収益化をスタートできる。
  - **表示されるだけで報酬**: クリック報酬型だけでなく「表示報酬型」の広告もあり、PVがあれば着実にポイントが貯まる。
  - **高い自由度**: 2chまとめサイトやアダルトサイトなど、他社では断られるジャンルでも掲載可能である。
</Card>

## 2. 広告タグの発行手順

まずは広告の「部品」となるタグを取得する。

<Steps>

1.  **公式サイトへ登録**
    [忍者AdMax](https://adf.shinobi.jp/r/6e628df1eff1a4679472c26163d8cbcf)にアクセスし、アカウントを作成する。

2.  **広告枠の作成**
    管理画面から「新しい広告枠を作る」を選択する。
    - **サイト属性**: 「PC用」または「スマートフォン用」を選択（レスポンシブ広告が推奨）。
    - **サイトURL**: 構築したサイトのURLを入力する。

3.  **広告タグをコピー**
    発行された `<script>` から始まるHTMLタグをコピーして控えておく。

</Steps>

## 3. Astro/Starlightへの埋め込み方法

Astroで広告を表示させるには、コンポーネント化するのが最もスマートである。

### 広告用コンポーネントの作成

プロジェクトの `src/components/` フォルダに `NinjaAd.astro` を作成し、以下のように記述する。

```astro title="src/components/NinjaAd.astro"
---
# 忍者AdMaxから取得したタグをここに貼り付けます
---
<div class="ad-container">
  <!-- ここにコピーしたscriptタグを貼り付け -->
  <script type="text/javascript" src="..."></script>
</div>

<style>
  .ad-container {
    margin: 2rem 0;
    display: flex;
    justify-content: center;
    overflow: hidden;
  }
</style>
```

### 記事（MDX）での使用方法

広告を表示したい場所でコンポーネントをインポートして呼び出すだけである。

```mdx title="your-article.mdx" ins={5, 9}
---
title: 私の記事
description: 記事の概要
---
import NinjaAd from '../../../components/NinjaAd.astro';

記事の内容...

<NinjaAd />

続きの内容...
```

## まとめ：まずは「500円」を目指そう

<Aside type="tip" title="最低支払額">
  忍者AdMaxの最低支払額は**500円**からと非常に低く設定されている。
</Aside>

「完全無料」で運用しているこのサイトなら、月500円の収益が出るだけで**完全な黒字**である。まずはタグを設置して、自分のコンテンツから初めての収益が発生する感動を味わってみよう！本サイトも既にAdMaxを導入しているが、コンテンツに魅力がないためか、まだ500円に届いていない(泣)。

## FAQ：忍者AdMaxと収益化のよくある質問

### Q: 広告を貼ることでサイトが重くならないか？
**A:** 多少のJavaScript読み込みが発生するが、Astroの高速なベース性能があるため、**一般的なブログよりはるかに軽快**に動作する。

### Q: 複数の広告を貼ってもよいか？
**A:** はい、可能である。ただし、記事の読みやすさを損なわないよう、**適切な配置** （記事下やサイドバーなど）を心がけること。

### Q: 報酬はどうやって受け取るのか？
**A:** 貯まったポイントは、**現金やPex**に交換可能である。500円分から交換できるため、モチベーションを維持しやすいのが特徴である。
