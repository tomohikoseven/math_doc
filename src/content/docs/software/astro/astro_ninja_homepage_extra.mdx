## 1. 忍者AdMaxが「最強の代替案」である理由

なぜ、Astro × 忍者ホームページの構成において忍者AdMaxが選ばれるのか、その魅力をまとめました。

<Card title="忍者AdMaxの3大メリット" icon="star">
  - **審査なしで即開始**: 登録したその日から広告を掲載し、収益化をスタートできます。
  - **表示されるだけで報酬**: クリック報酬型だけでなく「表示報酬型」の広告もあり、PVがあれば着実にポイントが貯まります。
  - **高い自由度**: 2chまとめサイトやアダルトサイトなど、他社では断られるジャンルでも掲載可能です。
</Card>

## 2. 広告タグの発行手順

まずは広告の「部品」となるタグを取得します。

<Steps>

1.  **公式サイトへ登録**
    [忍者AdMax](https://adf.shinobi.jp/r/6e628df1eff1a4679472c26163d8cbcf)にアクセスし、アカウントを作成します。

2.  **広告枠の作成**
    管理画面から「新しい広告枠を作る」を選択します。
    - **サイト属性**: 「PC用」または「スマートフォン用」を選択（レスポンシブ広告が推奨です）。
    - **サイトURL**: 構築したサイトのURLを入力します。

3.  **広告タグをコピー**
    発行された `<script>` から始まるHTMLタグをコピーして控えておきます。

</Steps>

## 3. Astro/Starlightへの埋め込み方法

Astroで広告を表示させるには、コンポーネント化するのが最もスマートです。

### 広告用コンポーネントの作成

プロジェクトの `src/components/` フォルダに `NinjaAd.astro` を作成し、以下のように記述します。

```astro title="src/components/NinjaAd.astro"
---
# 忍者AdMaxから取得したタグをここに貼り付けます
---
<div class="ad-container">
  <!-- ここにコピーしたscriptタグを貼り付け -->
  <script type="text/javascript" src="..."></script>
</div>

<style>
  .ad-container {
    margin: 2rem 0;
    display: flex;
    justify-content: center;
    overflow: hidden;
  }
</style>
```

### 記事（MDX）での使用方法

広告を表示したい場所でコンポーネントをインポートして呼び出すだけです。

```mdx title="your-article.mdx" ins={5, 9}
---
title: 私の記事
description: 記事の概要
---
import NinjaAd from '../../../components/NinjaAd.astro';

記事の内容...

<NinjaAd />

続きの内容...
```

## まとめ：まずは「500円」を目指そう

<Aside type="tip">
  忍者AdMaxの最低支払額は**500円〜**と非常に低く設定されています。
</Aside>

「完全無料」で運用しているこのサイトなら、月500円の収益が出るだけで**完全な黒字**です。まずはタグを設置して、自分のコンテンツから初めての収益が発生する感動を味わってみてください！本サイトも既にAdMaxを導入していますが、コンテンツに魅力がないためか、まだ500円に届いていません(泣)。

## FAQ：忍者AdMaxと収益化のよくある質問

### Q: 広告を貼ることでサイトが重くなりませんか？
**A:** 多少のJavaScript読み込みが発生しますが、Astroの高速なベース性能があるため、**一般的なブログよりはるかに軽快**に動作します。

### Q: 複数の広告を貼ってもいいですか？
**A:** はい、可能です。ただし、記事の読みやすさを損なわないよう、**適切な配置** （記事下やサイドバーなど）を心がけましょう。

### Q: 報酬はどうやって受け取りますか？
**A:** 貯まったポイントは、**現金やPex**に交換可能です。500円分から交換できるため、モチベーションを維持しやすいのが特徴です。
