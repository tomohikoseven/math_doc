DietPi（bookworm環境）でアプリケーションウィンドウをWindowsのようにショートカットキー（「**Windowsキー + 矢印**」など）でタイル配置するには、サードパーティ製ツールの `quicktile` を導入するのが手軽である。

初期状態のDietPi（デスクトップ環境による）では、このタイル配置のショートカットが標準で備わっていない場合があるため、このツールを利用する。

## QuickTileのインストール手順

<Steps>

1. 依存モジュールをインストールする。

   ```bash
   sudo apt install python3 python3-pip python3-setuptools python3-gi python3-xlib python3-dbus gir1.2-glib-2.0 gir1.2-gtk-3.0 gir1.2-wnck-3.0
   ```

1. ダウンロードツール（`git`）をインストールする。

   ```bash
   sudo apt install git
   ```

1. GitHubから `quicktile` のリポジトリをクローンする。

   ```bash
   git clone https://github.com/ssokolow/quicktile.git
   ```

1. クローンしたディレクトリに移動し、インストールスクリプトを実行する。

   ```bash
   cd quicktile
   ./install.sh
   ```

   <Aside type="caution" title="インストール時のエラーについて">
   実行時に `error: externally-managed-environment` というエラーメッセージが出力される場合があるが、動作には影響しないため無視して問題ない。
   </Aside>

</Steps>

## 動作確認

DietPiを再起動した後、「**Ctrl + Alt + テンキーの6**（右矢印に相当）」を押下することで、アクティブなウィンドウが画面右半分にタイル配置されるか確認する。

## まとめ

* DietPi標準ではウィンドウのタイル配置ショートカットがないため、`quicktile` を導入する。
* GitHubからソースコードを取得し、付属のインストールスクリプトで導入可能である。
* インストール・再起動後は、指定のショートカットキー（例: `Ctrl + Alt + テンキーの6`）でタイル配置が実行できる。
