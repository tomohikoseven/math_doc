# 【数学Ⅲ】「いろいろな関数の導関数」の基礎とは何か？ 〜関数ごとの公式とその性質〜

前の単元で「微分の計算ルール（道具）」を手に入れたら、次はその道具を使って、理系数学で頻出する「いろいろな関数」を実際に微分していきる。ここでは、各関数の微分公式を正確に暗記し、組み合わせて使えるようになることが基礎となる。

## 累乗の関数（分数・マイナス乗）の微分

実数乗に拡張された累乗の関数の微分公式により、分数関数や無理関数も統一的なルールで微分できるようになる。

数学Ⅱでは $y = x^2$ などの「自然数乗」の微分を学びましたが、数学Ⅲではこれが実数全体に拡張される。

<FlatList>
- <b>累乗の公式</b>：$y = x^alpha => y' = alpha x^(alpha - 1)$ （$alpha$ は実数）
</FlatList>

この公式により、$display(y = sqrt(x) = x^(1/2))$ の微分が $display(y' = 1 / (2 sqrt(x)))$ となったり、$display(y = 1/x = x^(-1))$ の微分が $display(y' = -1 / x^2)$ になったりすることが統一的に理解できる。

## 三角関数の微分

周期的な現象を表す三角関数の微分は、極限の公式に基づいて導出され、符号の変化に注意して公式を適用しる。

波動や円運動など、周期的な現象を表す三角関数の微分は以下の通りである。

<FlatList>
- <b>サインの微分</b>：$(sin x)' = cos x$
- <b>コサインの微分</b>：$(cos x)' = -sin x$ （マイナスがつくことに注意！）
- <b>タンジェントの微分</b>：$display((tan x)' = 1 / cos^2 x)$
</FlatList>

これらの公式は、極限の重要公式 $display(lim_(x -> 0) (sin x)/x = 1)$ を用いて導出される。導出過程自体が入試で問われることもあるため、丸暗記ではなく成り立ちを理解することが基礎となる。

## 指数関数と対数関数の微分

自然対数の底であるネイピア数を用いた微分公式を中心に、複雑な関数には対数微分法という強力な計算手法を活用しる。

自然現象の増減（人口増加や放射性物質の崩壊など）を表す指数・対数関数には、「ネイピア数 $e$（自然対数の底）」が深く関わりる。

<FlatList>
- <b>自然対数と自然指数</b>：$(e^x)' = e^x$, $display(quad (log x)' = 1/x)$（$e^x$ は「微分しても形が変わらない」という特別で美しい性質を持ちます）
- <b>一般の底の場合</b>：$(a^x)' = a^x log a$, $display(quad (log_a x)' = 1 / (x log a))$
</FlatList>

また、底と指数の両方に変数 $x$ が含まれる関数（例：$y = x^x$）を微分する際に用いる、両辺の対数をとってから微分する<b>「対数微分法」</b>というテクニックも、この単元の重要な基礎である。

## 高次導関数

関数を繰り返し微分して得られる高次導関数は、後の単元で曲線の凹凸や物理学の加速度を解析するための不可欠な概念である。

関数を1回微分したものを第1次導関数 $y'$、さらにそれをもう一度微分したものを第2次導関数 $y''$、さらに微分を重ねたものを第 $n$ 次導関数 $y^((n))$ と呼びる。
高次導関数は、この後に学ぶ「曲線の凹凸」や「物理学における加速度」などを考えるために必須の概念である。

## まとめ

「いろいろな関数の導関数」の基礎は、各関数の微分公式を瞬時に引き出し、適切に使いこなすことにある。

<FlatList>
- 累乗の微分公式を実数全体に拡張し、無理関数なども微分できるようにする
- 三角関数の微分公式を極限の成り立ちから理解し暗記する
- ネイピア数を用いた指数・対数関数の微分と、対数微分法を習得する
- 複数回の微分による高次導関数の計算方法と概念を把握する
</FlatList>
