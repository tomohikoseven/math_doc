# 【数学Ⅲ】「不定積分」の基礎とは何か？ 〜多様な関数を積分する強力な計算手法〜

数学Ⅲの積分法は、微分法の逆演算として、あらゆる形の関数を元に戻す（積分する）技術を習得することが第一歩となります。ここでは、個別の関数の積分公式に加え、複雑な関数の積分を可能にするための2大ツールである「置換積分法」と「部分積分法」の基礎を解説します。

## 不定積分の基本公式

数学Ⅲでは多項式だけでなく、分数関数や三角関数、指数関数といった様々な関数の不定積分公式を暗記し使いこなす必要があります。

数学Ⅱまでとは異なり、様々な形の関数を積分するための公式が登場します。これらはすべて「微分すると元に戻る関数」を見つけるプロセスから導かれます。

<FlatList>
- **累乗の公式**：$\int x^\alpha dx = \frac{x^{\alpha+1}}{\alpha+1} + C$ （$\alpha \neq -1$ の実数）。$\alpha = -1$ の場合は $\int \frac{1}{x} dx = \log|x| + C$ となります。
- **三角関数の公式**：$\int \sin x dx = -\cos x + C$ など、微分の公式を逆向きに適用します。
- **指数関数の公式**：$\int e^x dx = e^x + C$ や、一般の底の場合は $\int a^x dx = \frac{a^x}{\log a} + C$ となります。
</FlatList>

## 複雑な関数を解く「置換積分法」

関数の中に別の関数が入り込んでいる場合、変数の一部を別の文字に置き換える置換積分法を用いることで、積分可能な形に変換します。

合成関数の微分の逆演算にあたるのが「置換積分法」です。そのままでは積分できない複雑な式を、変数 $x$ を別の変数 $t$ （例えば $x = g(t)$ や $t = f(x)$ など）に置き換えることでシンプルな式に変形します。

<FlatList>
- **置き換えの手順**：式の中の複雑な部分を $t$ と置き、両辺を微分して $dx$ と $dt$ の関係式（$dx = g'(t)dt$）を求めます。
- **分数型の置換積分**：$\int \frac{f'(x)}{f(x)} dx = \log|f(x)| + C$ という特別な公式は、置換積分の代表的な応用例として非常によく使われます。
</FlatList>

## 積の形を解く「部分積分法」

異なる種類の関数が掛け合わされている場合、微分法の積の公式を逆算した部分積分法を利用して積分を計算します。

「多項式 $\times$ 三角関数」や「多項式 $\times$ 指数関数」のように、全く異なる種類の関数が掛け算されている場合に威力を発揮するのが「部分積分法」です。

<FlatList>
- **部分積分の公式**：$\int f(x)g'(x) dx = f(x)g(x) - \int f'(x)g(x) dx$
- **関数の選び方**：どちらを「微分する側（$f(x)$）」にし、どちらを「積分する側（$g'(x)$）」にするかの見極めが重要です。一般に、対数関数は微分側に、指数・三角関数は積分側に選ぶとスムーズに計算できます。
</FlatList>

## まとめ

不定積分の基礎は、基本公式の暗記と、置換積分・部分積分の使い分けを瞬時に判断できる計算力を養うことにあります。

<FlatList>
- 多様な関数の不定積分公式を正確に暗記し、瞬時に引き出せるようにする
- 関数の形から置換積分法を使うべきタイミングを見抜き、正確に変数変換を行う
- 異なる種類の関数の積に対して、部分積分法を適用して計算を簡略化する
- これらの方針決定を迷わず行えるようになるまで計算練習を反復する
</FlatList>
