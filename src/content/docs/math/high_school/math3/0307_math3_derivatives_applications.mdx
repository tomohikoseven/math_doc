# 【数学Ⅲ】「導関数の応用」の基礎とは何か？ 〜微分の図形的な意味と強力な定理〜

複雑な関数の微分ができるようになったら、次はその「導関数（$y'$）」を何に使うのかを学びます。微分法の応用は大きく2つの段階に分かれていますが、その前半戦である「導関数の応用」では、**「微分係数がグラフの接線の傾きを表す」**という図形的な基礎を徹底的に活用します。

## 接線と法線の方程式

導関数の値がその点における接線の傾きになるという性質を利用し、接線とそれに垂直な法線の方程式を導き出します。

ある関数 $y = f(x)$ のグラフ上にある点（接点） $(a, f(a))$ における接線を引くことを考えます。
導関数の値 $f'(a)$ は、まさに「その点における接線の傾き」を表しています。したがって、接線の方程式は数学Ⅱの図形と方程式の知識を使って以下のように表せます。

<FlatList>
- **接線の方程式**：$y - f(a) = f'(a)(x - a)$
- **法線の方程式**：$y - f(a) = -\frac{1}{f'(a)}(x - a)$ （ただし $f'(a) \neq 0$）
</FlatList>

接点において接線と垂直に交わる直線を「法線」と呼び、垂直な2直線の傾きの積が $-1$ になる性質を利用して式を立てます。これらは、数学Ⅲの図形問題を解くための最も基本的な武器となります。

## 共通接線

2つの異なる曲線に同時に接する共通接線は、接点を共有するケースと離れた接点を持つケースで分けて立式します。

2つの異なる曲線 $y = f(x)$ と $y = g(x)$ があるとき、両方に接する直線を「共通接線」と呼びます。これには以下の2つのケースがあり、アプローチが異なります。

<FlatList>
- **2曲線が接する場合**：接点 $x=t$ を共有し、その点での高さと傾きが等しい（$f(t)=g(t)$ かつ $f'(t)=g'(t)$）という条件を用います。
- **離れた2曲線に接線を引く場合**：一方の接点 $x=s$ と、もう一方の接点 $x=t$ における接線の方程式が「全く同じ直線になる」という条件から式を立てます。
</FlatList>

## 平均値の定理（数学の裏ボス）

平均値の定理は、関数の区間全体の変化と特定の一点における接線の傾きを結びつけ、不等式の証明や関数の増減の根拠を与えます。

この単元で最も論理的で、かつ奥が深い基礎が**「平均値の定理」**です。

<FlatList>
- **平均値の定理の主張**：連続で滑らかな関数のグラフにおいて、2点A, Bを結んだ直線の傾きと「同じ傾きを持つ接線」が、AとBの間のどこかに必ず引ける。
- **数式表現**：$\frac{f(b)-f(a)}{b-a} = f'(c)$ を満たす $c$ が $a$ と $b$ の間に存在する。
</FlatList>

この定理は、具体的な値を求める計算問題よりも、**「不等式の証明」**や**「関数の値の変化を導関数 $f'(x)$ の情報から評価する」**といった論証問題で絶大な威力を発揮します。「ある区間で $f'(x)>0$ ならば、その区間で関数は単調に増加する」という事実も、実はこの平均値の定理によって厳密に証明されています。

## 結論

「導関数の応用」の基礎は、単なる計算だった微分を図形的な傾きとして結びつけ、論理的思考の土台を作ることです。

<FlatList>
- 導関数を利用してグラフの接線と法線の方程式を正確に求める
- 2つの曲線が接する条件や共通接線の求め方を理解する
- 平均値の定理の意味を把握し、不等式の証明などに活用する
</FlatList>
