　微分計算が一定のルールに従って機械的に行えるのに対し、積分計算（不定積分の決定）は「微分の逆演算」という性質上、より多角的で戦略的なアプローチを必要とする。すべての関数が初等関数で積分できるわけではないが、数学や物理学で頻出する積分の多くは、適切な**変形**と**テクニック**を駆使することで、既知の基本公式へと帰着させることが可能である。

　本稿では、積分計算の「王道」である置換積分や部分積分から、特定の複雑な関数に対して威力を発揮する特殊な置換法まで、体系的な計算手法を解説する。

## 2. 置換積分法 (Integration by Substitution)

　置換積分法は、積分変数を別の変数に置き換えることで、複雑な積分を**基本公式が適用できる形に変形する手法**である。
- **基本原理**：合成関数の微分の逆演算。
- **注意点**：変数の置き換えに伴い、必ず**微分の変換（$d x -> d t$）**と、定積分の場合は**積分範囲の変更**を行う必要がある。

### 置換積分の公式
　関数 $x = phi(t)$ が微分可能であるとき、
$$
integral f(x) d x = integral f(phi(t)) phi'(t) d t.
$$
定積分の場合は、積分範囲に考慮して
$$
integral_a^b f(x) d x = integral_alpha^beta f(phi(t)) phi'(t) d t quad (a=phi(alpha), b=phi(beta)).
$$

### 置き換えのコツ
　何を変数 $t$ と置くべきか迷った際は、以下の3点を意識すると見通しが良くなる。

1. **「微分した形」が近くにあるものを探す**: 
   $integral f(g(x)) g'(x) d x$ のように、ある部分を微分したものが掛けられている場合、その部分（$g(x)$）を $t$ と置く。
2. **「中身」をひとまとめにする**: 
   累乗の中身 $(a x + b)^n$、ルートの中身 $sqrt(f(x))$、指数の肩 $e^(f(x))$ などを $t$ と置く。
3. **分母を丸ごと置いてみる**: 
   分母を $t$ と置くことで、$integral (f'(x))/(f(x)) d x = log |f(x)| + C$ の形に帰着できることが非常に多い。

### 具体例

**例題 1（「微分した形」を探す例）：** $integral x e^(x^2) d x$

$t = x^2$ とおくと、$d t = 2x d x$ より $x d x = 1/2 d t$ である。
$$
integral x e^(x^2) d x = integral e^t dot 1/2 d t = 1/2 e^t + C = 1/2 e^(x^2) + C
$$

**例題 2（「中身」を置き換える例）：** $integral (3x+2)^4 d x$

$t = 3x+2$ とおくと、$d t = 3 d x$ より $d x = 1/3 d t$ である。
$$
integral (3x+2)^4 d x = integral t^4 dot 1/3 d t = 1/15 t^5 + C = 1/15 (3x+2)^5 + C
$$

**例題 3（分母を $t$ と置く例）：** $integral x/(x^2+1) d x$

$t = x^2+1$ とおくと、$d t = 2x d x$ より $x d x = 1/2 d t$ である。
$$
integral x/(x^2+1) d x = integral (1/2 d t)/t = 1/2 log |t| + C = 1/2 log (x^2+1) + C
$$

## 3. 部分積分法 (Integration by Parts)
　部分積分法は、**積の微分公式を逆に利用した手法**であり、**2つの関数の積を積分する際に強力な武器**となる。基本原理は積の微分法の逆演算であり、一方の関数を微分し、もう一方を積分することで、計算をより容易な積分に帰着させる。活用のコツは、2つの関数のうち、どちらを「微分する側 ($u$)」とし、どちらを「積分する側 ($v'$)」とするかの選択をすることである。

### 部分積分の公式
　$u, v$ が微分可能な関数であるとき、
$$
integral u v' d x = u v - integral u' v d x.
$$
定積分の場合は、
$$
integral_a^b u v' d x = [u v]_a^b - integral_a^b u' v d x.
$$

#### 部分積分適用の判断基準
　どのような場合に部分積分を検討すべきか、主な目安は以下の通りである。

1. **種類の異なる関数の積**:
   「多項式関数 $times$ 指数関数」や「多項式関数 $times$ 三角関数」のように、微分すると次数が下がる関数と、積分しても形が大きく変わらない関数が組み合わさっている場合。
2. **直接の積分公式がない関数**:
   $log x$ や $arctan x$ のように、微分は容易だが積分公式が未知の関数の場合。「$1$ との積」とみなして部分積分を適用する（例：$integral 1 dot log x d x$）。
3. **微分・積分を繰り返すと元の形が現れるもの**:
   $e^x sin x$ のように、複数回の手法適用によって元の積分が再び現れ、方程式として解ける場合。

#### どちらを $u$ に選ぶか
　一般的に、対数関数 (L) $arrow.r$ 多項式関数 (A) $arrow.r$ 三角関数 (T) $arrow.r$ 指数関数 (E) の順（LATEの法則）で、$u$ に選ぶと計算がスムーズになることが多い。

### 具体例

**例題 1（種類の異なる関数の積）：** $integral x e^x d x$

$u = x, v' = e^x$ とおくと、$u' = 1, v = e^x$ である。
$$
integral x e^x d x = x e^x - integral 1 dot e^x d x = x e^x - e^x + C = (x-1) e^x + C
$$

**例題 2（公式がない関数）：** $integral log x d x$

$u = log x, v' = 1$ とおくと、$u' = 1/x, v = x$ である。
$$
integral log x d x = integral 1 dot log x d x = x log x - integral 1/x dot x d x = x log x - x + C
$$

**例題 3（元の形が現れるもの）：** $integral e^x sin x d x$

$I = integral e^x sin x d x$ とおく。$u = sin x, v' = e^x$ とすると $u' = cos x, v = e^x$ より、
$$
I = e^x sin x - integral e^x cos x d x
$$
さらに、右辺の積分に対して $u = cos x, v' = e^x$ とすると $u' = -sin x, v = e^x$ より、
$$
I = e^x sin x - (e^x cos x - integral e^x (-sin x) d x) = e^x (sin x - cos x) - I
$$
これより、$2I = e^x (sin x - cos x)$ となり、積分定数を加えて $I = 1/2 e^x (sin x - cos x) + C$ を得る。

## 4. 有理関数の積分 (Partial Fraction Decomposition)
　分母が因数分解できる有理関数 $display(P(x)/Q(x))$ は、**部分分数分解**を行うことで、対数関数や逆正接関数（$arctan$）の積分に帰着させることができる。

### 手順
1. **分母の因数分解**：分母 $Q(x)$ を実数の範囲で（1次式や2次式の積に）因数分解する。
2. **部分分数への展開**：適切な係数 $A, B, dots$ を用いて展開する。
3. **各項の積分**：$display(1/(x-a)^n)$ や $display((A x + B)/(x^2 + p x + q))$ の形を積分する。

### 具体例
**例題：** $integral 1/(x^2 - x) d x$

1. **分母の因数分解**：
   分母を $x^2 - x = x(x-1)$ と因数分解する。
2. **部分分数への展開**：
   $display(1/(x(x-1)) = A/x + B/(x-1))$ とおいて恒等式を解くと、$A = -1, B = 1$ となり、
   $$ 1/(x^2 - x) = 1/(x-1) - 1/x $$
   を得る。
3. **各項の積分**：
   $$
   integral 1/(x^2 - x) d x = integral ( 1/(x-1) - 1/x ) d x = log |x-1| - log |x| + C = log |(x-1)/x| + C
   $$

## 5. 特殊な置換法
　特定の形を含む積分では、経験的に有効な置換が知られている。

### (1) 三角関数の有理関数 (Weierstrass置換)
　$t = tan(display(x/2))$ と置くことで、あらゆる三角関数の有理関数を $t$ の有理関数の積分に変換できる。例えば、
$$
sin x = display((2t)/(1+t^2)), quad cos x = display((1-t^2)/(1+t^2)), quad d x = display((2 d t)/(1+t^2))
$$

#### 具体例
**例題：** $integral 1/(1 + sin x) d x$

$t = tan(display(x/2))$ と置くと、
$$
integral 1/(1 + sin x) d x = integral display(1/(1 + (2t)/(1+t^2))) dot display((2 d t)/(1+t^2)) = integral display((1+t^2)/(1+t)^2) dot display((2 d t)/(1+t^2)) = integral 2/(1+t)^2 d t
$$
これを計算して $t$ を戻すと、
$$
- 2/(1+t) + C = - 2/(1 + tan(display(x/2))) + C
$$
を得る。

### (2) 無理関数の積分
　$sqrt(a x^2 + b x + c)$ を含む積分では、以下の置換が有効である。

- **平方完成による基本形への帰着**：
  - $sqrt(a^2 - x^2) quad arrow.r quad x = a sin theta$
  - $sqrt(x^2 + a^2) quad arrow.r quad x = a tan theta$
  - $sqrt(x^2 - a^2) quad arrow.r quad x = display(a / cos theta)$ または $x = a cosh t$
- **オイラー置換 (Euler Substitution)**：
$$ sqrt(a x^2 + b x + c) = t - sqrt(a) x quad (a > 0 " のとき") $$


#### 具体例
##### 平方完成による基本形への帰着
**例題：** $integral 1/(sqrt(x^2 + 2x + 2)) d x$

分母の中身を平方完成すると $x^2 + 2x + 2 = (x+1)^2 + 1^2$ となる。
ここで $x+1 = tan theta$ と置換すると、$d x = display(1/(cos^2 theta)) d theta$ および $sqrt((x+1)^2 + 1) = display(1/(cos theta))$ より、
$$
integral 1/(sqrt(x^2 + 2x + 2)) d x = integral cos theta dot 1/(cos^2 theta) d theta = integral 1/(cos theta) d theta
$$
となり、基本形の積分に帰着できる。最終的に次を得る。
$$
log |x+1 + sqrt(x^2 + 2x + 2)| + C
$$

##### オイラー置換
**例題：** $integral 1/(sqrt(x^2 + 1)) d x$

$sqrt(x^2 + 1) = t - x$ と置くと、両辺を2乗して $x^2 + 1 = t^2 - 2 t x + x^2$ より $x = (t^2 - 1)/(2t)$ である。
これを微分すると $d x = display((t^2 + 1)/(2t^2)) d t$ となり、
また $sqrt(x^2 + 1) = t - x = display((t^2 + 1)/(2t))$ である。これらを代入すると、
$$
integral 1/(sqrt(x^2 + 1)) d x = integral display((2t)/(t^2 + 1)) dot display((t^2 + 1)/(2t^2)) d t = integral 1/t d t = log |t| + C
$$
$t = x + sqrt(x^2 + 1)$ を戻して、以下の結果を得る。
$$
log |x + sqrt(x^2 + 1)| + C
$$


## まとめ
　複雑な積分を解く際の思考プロセスは、「**いかにして基本公式が適用できる形へ帰着させるか**」に集約される。計算方針に迷った際は、以下の優先順位で検討すると見通しが良くなる。

1. **基本公式の確認**: 暗記している公式で即答できないか、定数倍や和の性質で分解できないか。
2. **置換積分の検討**: 「関数の塊の微分」が外側に掛かっていないか（$f(g(x))g'(x)$ の形）。
3. **部分積分の適用**: 「多項式 $times$ 指数・三角関数」や「対数関数（$1$ との積とみなす）」の形ではないか。
4. **式の変形（代数的処理）**: 有理関数の部分分数分解や、三角関数の加法定理・半角の公式などで簡略化できないか。
5. **特殊な置換法の活用**: それでも突破できない場合に、Weierstrass置換やオイラー置換などの強力な手法を検討する。

積分計算の習熟には、これらの手法を「どの順序で試すべきか」という直感を、多くの演習を通じて養うことが不可欠である。
