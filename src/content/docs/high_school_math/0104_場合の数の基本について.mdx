　「場合の数」という単元は，高校数学Aで学ぶ重要な単元の１つである．
これはある条件を満たすものの数を１つ１つ素朴に数えるのではなく，
もれなく・重複なく論理的に数える方法を学ぶ単元である．

　この「場合の数」は確率を計算する上で必須の知識である．またコンピュータのアルゴリズムを考える上でも役に立つ．
さらに論理的な思考力を養う上でも重要である．この単元がわかると多くのメリットがある．

## 場合の数の基本は何か．
　「場合の数」における基本は，
- 樹形図（順列）でものを数えられる
- 樹形図をもとに，以下の4つの法則を使ってものを数えられる
  1. 積の法則
  1. 和の法則
  1. 包除原理
  1. 重複度による重複の除去

ことである．これらが理解できれば，応用問題はこの組み合わせで解くことができる．

### 樹形図（順列）で数える．
　場合の数を数える上で、最も基本的な方法が樹形図(順列)である．すべての場合を網羅的に数えることができる．
数え方のルール（辞書式など）を決めれば，もれなく・重複なく数え上げることができる．

　例えば，「1,2の数字を使って3桁の整数は何個作れるか？」という問題を樹形図で考えてみる．
すべて書くと長くなるので，100の位が1である3桁の整数を表す樹形図を下の図に示す．

```d2 width="500" title="1,2を使った3桁の整数の100の位が1の場合の樹形図"
direction: right
100: {
  label: "1"
}
110: {
  label: "1"
}
120: {
  label: "2"
}
111: {
  label: "1"
}
112: {
  label: "2"
}
121: {
  label: "1"
}
122: {
  label: "2"
}
100 -- 110 -- 111
110 -- 112
100 -- 120 -- 121
120 -- 122
```
　この樹形図では4つの3桁の整数があることがわかり，数えることができた．
樹形図は、すべての場合を網羅的に書き出すことで、場合の数を数えることができる．
