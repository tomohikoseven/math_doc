<Card title="問題8(1)">
　次の行列式を計算せよ．
$$
mat(delim: "|", a, b, c, d;
b, a, d, c;
c, d, a, b;
d, c, b, a;)
$$
</Card>

## 解答

$$
mat(delim: "|", a, b, c, d;
b, a, d, c;
c, d, a, b;
d, c, b, a;)
&=
mat(delim: "|", a, b, c, d;
b, a, d, c;
a + c, b + d, a + c, b + d;
b + d, a + c, b + d, a + c;) \
&=
mat(delim: "|", a - c, b - d, c, d;
b - d, a - c, d, c;
0, 0, a + c, b + d;
0, 0, b + d, a + c;) \
&=
mat(delim: "|", a - c, b - d;
b - d, a - c;)
mat(delim: "|", a + c, b + d;
b + d, a + c;)
$$

- (1)は3行目に1行目を加え，4行目に1行目を加えた．
- (2)は1列目に3列目x(-1)を加え，2列目に4列目x(-1)を加えた．
- (3)は定理4.6.2(ブログ未記事)の展開をした．

$$
mat(delim: "|", a - c, b - d;
b - d, a - c;)
&=
mat(delim: "|", a - c, a + b - c - d;
b - d, a + b - c - d;) \
&=
mat(delim: "|", a - b - c + d, 0;
b - d, a + b - c - d;) \
&=
(a + b - c - d)(a - b - c + d)
$$

- (4)は2列目に1列目を加えた．
- (5)は1行目に2行目x(-1)を加えた．
- (6)は行列式を展開した．

$$
mat(delim: "|", a + c, b + d;
b + d, a + c;)
&=
mat(delim: "|", a + c, a + c + b + d;
b + d, a + b + c + d;) \
&=
mat(delim: "|", a - b + c - d, 0;
b + d, a + b + c + d;) \
&=
(a + b + c + d)(a - b + c - d)
$$

- (7)は2列目に1列目を加えた．
- (8)は1行目に2行目x(-1)を加えた．
- (9)は行列式を展開した．

最終的には，(3)，(6)，(9)を合わせて，
$$
mat(delim: "|", a, b, c, d;
b, a, d, c;
c, d, a, b;
d, c, b, a;)
=
(a + b + c + d)(a - b + c - d)(a + b - c - d)(a - b - c + d)
$$
