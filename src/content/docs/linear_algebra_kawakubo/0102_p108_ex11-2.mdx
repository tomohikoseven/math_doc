<Card title="問題11(2)">
　次の等式を証明せよ．
$$
mat(delim: "|", x, a_2, a_3, dots.c, a_n, 1;
a_1, x, a_3, dots.c, a_n, 1;
a_1, a_2, x, dots.c, a_n, 1;
dots.v, dots.v, dots.v, dots.down, dots.v, dots.v;
a_1, a_2, a_3, dots.c, x, 1;
a_1, a_2, a_3, dots.c, a_n, 1;)
=
product_(i = 1)^n (x - a_i)
$$
</Card>

## 解答
$$
mat(delim: "|", x, a_2, a_3, dots.c, a_n, 1;
a_1, x, a_3, dots.c, a_n, 1;
a_1, a_2, x, dots.c, a_n, 1;
dots.v, dots.v, dots.v, dots.down, dots.v, dots.v;
a_1, a_2, a_3, dots.c, x, 1;
a_1, a_2, a_3, dots.c, a_n, 1;)
&=
mat(delim: "|", x - a_1, 0, 0, dots.c, 0, 0;
0, x - a_2, 0, dots.c, 0, 0;
0, 0, x - a_3, dots.c, 0, 0;
dots.v, dots.v, dots.v, dots.down, dots.v, dots.v;
0, 0, 0, dots.c, x - a_n, 0;
a_1, a_2, a_3, dots.c, a_n, 1;) \
&=
(-1)^(2 n + 2)
mat(delim: "|", x - a_1, 0, 0, dots.c, 0;
0, x - a_2, 0, dots.c, 0;
0, 0, x - a_3, dots.c, 0;
dots.v, dots.v, dots.v, dots.down, dots.v;
0, 0, 0, dots.c, x - a_n ;) \
&=
product_(i = 1)^n (x - a_i)
$$

- (1)は1行目にn+1行目x(-1)を加える．$i$行目にn+1x(-1)を加える．$2 <= i <= n$．
- (2)は$n + 1$列目で余因子展開をした．
- (3)は対角行列式を展開し，整理した．
