import { Card } from "@astrojs/starlight/components";

<Card title="問題11(5)">
　次の等式を証明せよ．
$$
mat(delim: "|", 1 + x^2, x, 0, dots.c, 0;
x, 1 + x^2, x, dots.down, dots.v;
0, x, dots.down, dots.down, 0;
dots.v, dots.down, dots.down, 1 + x^2, x;
0, dots.c, 0, x, 1 + x^2 ;)
=
1 + x^2 + x^4 + dots.c + x^(2 n)
$$
</Card>

## 解答
　帰納法を用いる．
$$
A_n =
mat(delim: "|", 1 + x^2, x, 0, dots.c, 0;
x, 1 + x^2, x, dots.down, dots.v;
0, x, dots.down, dots.down, 0;
dots.v, dots.down, dots.down, 1 + x^2, x;
0, dots.c, 0, x, 1 + x^2 ;)
$$
とおく．$A_1 = 1 + x^2$，
$$
A_2
&=
mat(delim: "|", 1 + x^2, x;
x, 1 + x^2 ;) \
&= (1 + x^2)(1 + x^2) - x^2 \
&= 1 + x^2 + x^4
$$
であり，
$$
A_3
&=
mat(delim: "|", 1 + x^2, x, 0;
x, 1 + x^2, x;
0, x, 1 + x^2 ;) \
&=
(1 + x^2) A_2 - x
mat(delim: "|", x, 0;
x, 1 + x^2 ;) \
&=
(1 + x^2) A_2 - x^2 A_1 \
&=
1 + x^2 + x^4 + x^6
$$
である．

　つまり，$A_(n - 1)$まで題意が成り立つと仮定すると，
$$
A_n
&= (1 + x^2) A_(n - 1) - x^2 A_(n - 2) \
&= (1 + x^2)(1 + x^2 + x^4 + dots.c + x^(2(n - 1))) - x^2 (1 + x^2 + x^4 + dots.c + x^(2(n - 2))) \
&= 1 + x^2 + x^4 + dots.c + x^(2 n) "．"
$$

以上により，帰納法から題意が示せた．
