<Card title="問題7(4)">
　次の行列式を計算せよ．
$$
mat(delim: "|", a + b + c, - c, - b;
- c, a + b + c, - a;
- b, - a, a + b + c)
$$
</Card>

## 解答
$$
mat(delim: "|", a + b + c, - c, - b;
- c, a + b + c, - a;
- b, - a, a + b + c)
&=
mat(delim: "|", a + b, - c, - b;
a + b, a + b + c, - a;
-(a + b), - a, a + b + c) \
&=
(a + b)
mat(delim: "|", 1, - c, - b;
1, a + b + c, - a;
- 1, - a, a + b + c) \
$$
- (1)は2列目を1列目に加えた．
- (2)は1列目の共通因子(a+b)を外に出した．

$$
mat(delim: "|", 1, - c, - b;
1, a + b + c, - a;
- 1, - a, a + b + c)
&=
mat(delim: "|", 1, - c, - b;
0, a + b + 2 c, - a + b;
0, -(a + c), a + c) \
&=
mat(delim: "|", a + b + 2 c, - a + b;
-(a + b), a + c) \
&=
(a + c)
mat(delim: "|", a + b + 2 c, - a + b;
- 1, 1) \
&=
(a + c)
mat(delim: "|", 2(b + c), - a + b;
0, 1) \
&=
2(a + c)(b + c)
$$

- (3)は2行目に1行目x(-1)を加え，3行目に1行目を加えた．
- (4)は1列目で余因子展開をした．
- (5)は2行目の共通因子を外に出した．
- (6)は2列目を1列目に加えた．
- (7)は2行目で余因子展開をした．

(2)と(7)を合わせて，
$$
mat(delim: "|", a + b + c, - c, - b;
- c, a + b + c, - a;
- b, - a, a + b + c)
=
2(a + b)(b + c)(c + a)
$$
