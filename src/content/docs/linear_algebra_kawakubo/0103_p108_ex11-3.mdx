<Card title="問題11(3)">
　次の等式を証明せよ．
$$
mat(delim: "|", 1, 1, 1, dots.c, 1;
b_1, a_1, a_1, dots.c, a_1 ;
b_2, b_2, a_2, dots.c, a_2 ;
dots.v, dots.v, dots.down, dots.down, dots.v;
b_(n - 1), b_(n - 1), dots.c, b_(n - 1), a_(n - 1);)
=
product_(i = 1)^(n - 1)(a_i - b_i)
$$
</Card>

## 解答
$$
mat(delim: "|", 1, 1, 1, dots.c, 1;
b_1, a_1, a_1, dots.c, a_1 ;
b_2, b_2, a_2, dots.c, a_2 ;
dots.v, dots.v, dots.down, dots.down, dots.v;
b_(n - 1), b_(n - 1), dots.c, b_(n - 1), a_(n - 1);)
&=
mat(delim: "|", 1, 0, 0, dots.c, 0;
b_1, a_1 - b_1, 0, dots.c, 0;
b_2, 0, a_2 - b_2, dots.c, 0;
dots.v, dots.v, dots.down, dots.down, dots.v;
b_(n - 1), 0, dots.c,, a_(n - 1) - b_(n - 1);) \
&=
product_(i = 1)^(n - 1)(a_i - b_i)
$$

- (1)は$n$列目に$n - 1$列目x(-1)を加える．$n - 1$列目に$n - 2$列目x(-1)を加える．$dots.c$，2列目に1列目x(-1)を加える．
- (2)は下三角行列式を展開した．
