<Card title="問題10(2)">
　次の等式を証明せよ．
$$
mat(delim: "|", 1, 2, 3, dots.c, n;
- 1, 0, 3, dots.c, n;
- 1, - 2, 0, dots.c, n;
dots.v, dots.v, dots.v, dots.down, dots.v;
- 1, - 2, - 3, dots.c, 0;)
=
n!
$$
</Card>

## 解答
$$
mat(delim: "|", 1, 2, 3, dots.c, n;
- 1, 0, 3, dots.c, n;
- 1, - 2, 0, dots.c, n;
dots.v, dots.v, dots.v, dots.down, dots.v;
- 1, - 2, - 3, dots.c, 0;)
&=
mat(delim: "|", 1, 2, 3, dots.c, n;
0, 2, 6, dots.c, 2 n;
0, 0, 3, dots.c, 2 n;
dots.v, dots.v, dots.v, dots.down, dots.v;
0, 0, 0, dots.c, n;) \
&=
n!
$$

- (1)は1行目を$i$行目に加えた．$2 <= i <= n$．すると上三角行列になる．
- (2)は上三角行列式を展開した結果である．
