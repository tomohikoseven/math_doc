　次の行列式を計算せよ．
$$
mat(delim: "|", a, b, c, d;
- b, a, - d, c;
- c, d, a, - b;
- d, - c, b, a;)
$$
</Card>

## 解法
$$
A =
mat(delim: "(", a, b, c, d;
- b, a, - d, c;
- c, d, a, - b;
- d, - c, b, a;)
$$
とおく．
$$
A A^top
&=
mat(delim: "(", a^2 + b^2 + c^2 + d^2, 0, 0, 0;
0, a^2 + b^2 + c^2 + d^2, 0, 0;
0, 0, a^2 + b^2 + c^2 + d^2, 0;
0, 0, 0, a^2 + b^2 + c^2 + d^2 ;) \
&=
(a^2 + b^2 + c^2 + d^2)^4
$$
であることに気づけば，$d e t(A A^top) = d e t A "・" d e t(A^top) = d e t A "・" d e t A = (d e t A)^2$から，
$$
d e t A = (a^2 + b^2 + c^2 + d^2)^2
$$
