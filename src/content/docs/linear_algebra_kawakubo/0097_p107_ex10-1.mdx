## 解答
$$
mat(delim: "|", 1, 1, 1, dots.c, 1;
1, 2, 2, dots.c, 2;
1, 2, 3, dots.c, 3;
dots.v, dots.v, dots.v, dots.down, dots.v;
1, 2, 3, dots.c, n;)
&=
mat(delim: "|", 1, 1, 1, dots.c, 1;
0, 1, 1, dots.c, 1;
0, 1, 2, dots.c, 2;
dots.v, dots.v, dots.v, dots.down, dots.v;
0, 1, 2, dots.c, n - 1;) \
&=
mat(delim: "|", 1, 1, 1, dots.c, 1;
1, 2, 2, dots.c, 2;
1, 2, 3, dots.c, 3;
dots.v, dots.v, dots.v, dots.down, dots.v;
1, 2, 3, dots.c, n - 1;)
$$

- (1)は$i$行目に1行目x(-1)を加えた．$2 <= i <= n$．
- (2)は1列目に関して，余因子展開をした．

　つまり
$$
A_n =
mat(delim: "(", 1, 1, 1, dots.c, 1;
1, 2, 2, dots.c, 2;
1, 2, 3, dots.c, 3;
dots.v, dots.v, dots.v, dots.down, dots.v;
1, 2, 3, dots.c, n;)
$$
とおけば，$|A_n | =|A_(n - 1)| = dots.c =|A_1 | = 1$となる．
