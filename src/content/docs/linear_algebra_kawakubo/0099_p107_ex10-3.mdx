<Card title="問題10(3)">
　次の等式を証明せよ．
$$
mat(delim: "|", a + b, a, dots.c, a;
a, a + b, dots.c, a;
dots.v, dots.v,, dots.v;
a, a, dots.c, a + b;)
=
(n a + b) b^(n - 1)
$$
　(ただし，$n$は行列の次数)
</Card>

## 解答
$$
mat(delim: "|", a + b, a, dots.c, a;
a, a + b, dots.c, a;
dots.v, dots.v,, dots.v;
a, a, dots.c, a + b;)
&=
mat(delim: "|", n a + b, n a + b, dots.c, n a + b;
a, a + b, dots.c, a;
dots.v, dots.v,, dots.v;
a, a, dots.c, a + b;) \
&=
(n a + b)
mat(delim: "|", 1, 1, dots.c, 1;
a, a + b, dots.c, a;
dots.v, dots.v,, dots.v;
a, a, dots.c, a + b;) \
&=
(n a + b)
mat(delim: "|", 1, 1, dots.c, 1;
0, b, dots.c, 0;
dots.v, dots.v,, dots.v;
0, 0, dots.c, b;) \
&=
(n a + b) b^(n - 1)
$$

- (1)は1行目に2行目以下すべての行を加えた．
- (2)は1行目の共通因子を外出しした．
- (3)は1行目x(-a)を2行目以下すべての行に加えた．
- (4)は上三角行列式を展開した．
