#### 意味を持たせろ．
- 小学生まで → 意味のない文字列や絵合わせに対し記憶力を発揮。
- 中学生以上 → エピソード記憶が完成、つまり、論理だった記憶力が発達。
    - 具体的には，成長したから九九を覚えるのは至難の業ということ。そこに論理がないから．
- 回数を重ねて覚える方法では、そこに意味を伴う記憶でないなら、効果はうすい。
    - つまり，覚えにくく，持続性が伴わない．

#### 感情を利用せよ．
- LTP をおこす → $theta$リズムに乗る．つまり，感情を伴うと記憶しやすくなる．
    - 初めての体験
    - 覚えたい対象に興味を持つ（積極的な姿勢）
    - 感動
    - 情動：おもしろい、悲しいなど
    - 危機感 → 記憶力を一時的に増強
- 効率よく学習できるコツ → マンネリ化せず、適度な緊張感を保ちながら．

#### ストレスは記憶の敵
- 記憶力はストレスに弱い。 → ストレスがかかると LTP は大きく減弱。
    - 興味のない分野はやはり記憶しづらい．


#### シナプス可塑性の連合性という性質
- 事象と事象を連合させれば閾値よりも弱い刺激でLTPをおこせる性質
    - つまり、ものごとを互いに関連づければ（よく理解すれば）覚えやすくなる．

#### 忘却
- 覚えることは意図的にできるが、忘却は意図的にできない。
    - → エビングハウスは、4日経過すると、覚えなおすのに初めに覚えた時間の50%を要する．
    - → 忘却を早めることができる．他のことを追加して記憶すればいい．

<dl>
    <dt>記憶の干渉</dt>
    <dd>類似性があることを覚えると以前の記憶が妨げられてしまう．</dd>
</dl>

- 海馬は記憶の一時的な保管場所。期間は長くて1か月。
    - → 1か月以内に復習せよ。

#### 睡眠の効果
- 夢：睡眠中にあったことを思いおこすという行為。
    - → 脳の情報を整理、記憶を強化するための必須な過程．
    - → 記憶は夢を見ることで保存される．
    - → 覚えたその日に6時間以上眠ることが欠かせないという研究結果。

- 学習効果は眠らないと発揮されない？
    - → 直前に詰め込んだ知識よりも、覚えるのに数日かかった知識の方が、応用として利用しやすい。

#### 学習
- 脳は失敗を繰り返しながら記憶を作る。
    - → 試行錯誤することで記憶の強化につながる。
    - → 失敗は決してなくならないから、積極的に後悔するのではなく、反省することが大事。

- 独学は学習手順が大切。
    1. 大局を理解せよ。細かいことはその後に．
    1. 規則性を見つけ、理解せよ。
    1. 何度も練習を繰り返して覚えよう。

- カフェインは記憶力を促進する作用がある。
    - → ここぞというときに使う。耐性ができやすいため．

#### 結論
- 記憶力の増強は、脳の意識が大切．
