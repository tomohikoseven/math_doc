<Card title="問62">
　$lim_(n->oo) abs(u_(n+1)/u_n) = abs(a) < 1$ ならば $lim_(n->oo) u_n = 0$ であることを証明せよ．
</Card>

　$u_(n)$ とその前後の値が $abs(a) < 1$ の間隔に収まっていくなら，数列$u_n$ は$0$ に収まっていく．

## 証明の方針

- はさみうちの原理を使って $lim_(n->oo) abs(u_n) = 0$ を導き、[問39](/mcgraw_hill_college_exercise_series_calculus_upper/chapter03/q039/)から $lim_(n->oo) u_n = 0$ が示せる．
- はさみうちの原理で使う不等式は，$lim_(n->oo) abs(u_(n+1)/u_n) = abs(a) < 1$ という条件から導く．
    - 十分大きな $n$ に対して $abs(u_(n+1)/u_n)$ が1より小さい定数 $r$ で上から抑えられることを示す．

## 解答
　はさみうちの原理を使って $lim_(n->oo) abs(u_n) = 0$ 示す．

　仮定より，任意の正の数 $epsilon$ において，ある正の数 $delta(epsilon) $ が存在し，
$$
 n > delta(epsilon) => abs( abs(u_(n+1)/u_n) - abs(a) ) < epsilon
$$
が成り立つ．絶対値を外すことで，
$$
 abs(u_(n+1)/u_n) - abs(a) < epsilon \
 <=> abs(u_(n+1)/u_n) < abs(a) + epsilon
$$
となる．ここで $abs(a) < 1$ なので，$ abs(a) + epsilon_o < 1$ となる $epsilon_o$ を選べる(イプシロンデルタ論法はそういうことであるから．)．そのとき，$N_o = delta(epsilon_o)$ とおくと，
$$
 n > N_o => abs(u_(n+1)/u_n) < abs(a) + epsilon_o (< 1)
$$
となる．つまり，
$$
 n > N_o => abs(u_(n+1)) < (abs(a) + epsilon_o)abs(u_n)
$$
である．$n > N_o$ で成り立つので，
$$
 abs(u_(n+1)) < (abs(a) + epsilon_o)^(n-N_o+1) abs(u_(N_o))
$$
となる．

　$abs(a) + epsilon_o < 1$ なので，$lim_(n->oo) (abs(a) + epsilon_o)^(n-N_o+1) abs(u_(N_o)) = 0 $．$0 < abs(u_(n+1)) $ でもあるので，はさみうちの原理から $lim_(n->oo) abs(u_n) = 0$ が得られる．

　したがって，[問39](/mcgraw_hill_college_exercise_series_calculus_upper/chapter03/q039/)から$ lim_(n->oo) u_n = 0 $ となる．
