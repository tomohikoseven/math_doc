<Card title="問107">
　(a) $x$をその2乗が2よりも小さい有理数とするならば，$x +(2 - x^2) \/10$は$x$より大きいことを示せ．

　(b) $x$をその2乗が2より大きい有理数として，$x$より小さくて2乗が2より大きい有理数を，$x$を使って求めよ．
</Card>

　(a)は(b)を解くための問題だと思ったら違った．罠．

### (a)の解答
　題意を数式を使った命題に直すと，$x^2 < 2, thin x in QQ$ならば，$x < x + frac(2 - x^2, 10)$であることを示せということ．

$(x + frac(2 - x^2, 10)) - x = frac(2 - x^2, 10) > 0$なので，$x < x + frac(2 - x^2, 10)$．

### (b)の解答
　答えは(a)の$x + frac(2 - x^2, 10)$ではないか！？と考え，証明を試みたが違うことがわかった．
（(a)の意味は何なのかと落胆した．）

　題意を書き直す．$x^2 > 2, thin x in QQ$のとき，$y < x, thin y^2 > 2, thin y in QQ$なる$y$を$x$を使って求めよということ．

　__$x > sqrt(2)$のとき__，$y = frac(x^2 + 2, 2 x)$を考える．
$$
x - y = x - frac(x^2 + 2, 2 x) = frac(2 x^2 - x^2 - 2, 2 x) = frac(x^2 - 2, 2 x) > 0 "．"
$$
したがって，$y < x$．また，
$$
y^2 - 2 = (frac(x^2 + 2, 2 x))^2 - 2 = ("計算して") = (frac(x^2 - 2, 2 x))^2 > 0 "．"
$$
したがって，$y^2 > 2$．

　__$x < - sqrt(2)$のとき__，$y = x - 1$とおく．明らかに$y < x$である．
$$
2 - y^2 = 2 -(x - 1)^2 = ("計算して") = -(x - 1)^2 + 2 < 0 "．"
$$
したがって，$y^2 > 2$．

　以上をまとめると，
$$
y =
cases(frac(x^2 + 2, 2 x) & x > sqrt(2),
x - 1 & x < - sqrt(2))
$$


### コメント
　(b)は自力で解けなかった．Geminiに問題を与えて解いてもらった．
$frac(x^2 + 2, 2 x)$や$x - 1$のアイデアはGeminiが考えて出てきたものだ．
$x - y$の計算で，$x^2 > 2$を使うには$frac(x^2 + 2, 2 x)$なのだと思う．
