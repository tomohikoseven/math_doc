### 2025/01/13
- :white_check_mark: 数Ⅲ・C 入門問題精講 P65 練習問題9
  - 数列の極限は最低限以下の2つのことができればよい（これが基礎である）．
    1. 第$n$項を求める．
    2. n $\rightarrow \infty $の計算．
  - より発展したい場合，
    - 大学受験なら，公式を利用して解ける・公式の意味を理解する，効率的解法の知識を得ることをする． 
    - 数学をするなら，極限の意味・なぜ極限を計算するのか問う．
- :white_check_mark: 大学受験数学を解いてイラッとしてしまう理由がわかった．
  - **基礎的な解法で解いていないから**．
    例えば，実数$x$に対して，$x-1 < [x] \le x$であることを示せ に対して，$x-1 < [x]$ かつ $[x] \le x$を示すことが基本．
    しかし，$[x] \le x < [x]+1$を使って，左側の不等式と右側の不等式の変形から解いている．
