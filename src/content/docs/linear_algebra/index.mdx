　このブログの線形代数部分は線形代数学 (川久保勝夫著) で学んだ結果を整理して，線形代数の考え方を記述しているものである．

　しかし，この書籍に書かれていないことをブログ記事にしているものも多く存在する．書籍に書かれていることと違うという苦情は控えていただきたい．
これは私が線形代数をより深く考えた結果であるためだ．読者の方の選択肢は受け入れてもらうか，受け入れないでブログから離れるかである．

　このブログの線形代数部分は，ジョルダン標準形の証明をゴールとしている．
