<Card title="定理6.10.4">
　$V, W, U$をベクトル空間，$bold(v)_1, dots.c bold(v)_n$, $bold(w)_1, dots.c bold(w)_m$, $bold(u)_1, dots.c, bold(u)_l$をそれぞれ$V, W, U$の基底とする．２つの線形写像
$$
f colon V -> W, g colon W -> U
$$
が与えられたとき，上記基底に関して，$f, g$に対応する表現行列をそれぞれ$A, B$とすると，合成写像
$$
g circle.small f colon V -> U
$$
に対応する$"表現行列" C$は，$"等式" C = B A$を満たす．
</Card>

## 定理の主張
　合成写像の基底に関する表現行列は，合成している写像の表現行列の積で素直に表せられることを主張している．

## 証明の考え方
　$C = B A$を示すためには，行列Cと行列BAの成分どうしが同じであることを示せばよいが，**線形写像の行列表現と成分ベクトルとの関係性から示したほうが簡単そう**であるので，そうする．

　ベクトル空間$V$のベクトル$bold(x)$を基底$bold(v)_1, dots.c, bold(v)_n$に関する成分ベクトル$mat(delim: "(", x_1 ; dots.v; x_n)$，ベクトル空間$W$のベクトル$bold(y)$を基底$bold(w)_1, dots.c, bold(w)_m$に関する成分ベクトル$mat(delim: "(", y_1 ; dots.v; y_m)$，ベクトル空間$U$のベクトル$bold(z)$を基底$bold(u)_1, dots.c, bold(u)_l$に関する成分ベクトル$mat(delim: "(", z_1 ; dots.v; z_l)$とする．

　題意から，表現行列と成分ベクトルの関係式は
$$
mat(delim: "(", y_1 ; dots.v; y_m) = A mat(delim: "(", x_1 ; dots.v; x_n), mat(delim: "(", z_1 ; dots.v; z_l) = B mat(delim: "(", y_1 ; dots.v; y_m), mat(delim: "(", z_1 ; dots.v; z_l) = C mat(delim: "(", x_1 ; dots.v; x_n)
$$

左の2式から，
$$
mat(delim: "(", z_1 ; dots.v; z_l)
= B A
mat(delim: "(", x_1 ; dots.v; x_n)
$$
となる．したがって，一番右の関係式と比較することによって，$C = B A$となる．
