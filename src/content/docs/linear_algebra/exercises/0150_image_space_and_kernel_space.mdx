## 例題の主張
　線形写像が具体的に与えられたとき，$"Im" F "と" "Ker" F$が求められる．

## 解答の考え方
<Steps>

1. $"Im" F$の基底を１つ求める．

    　$"Im" F$の基底を１つ求める．すなわち，$"Im" F = L(bold(a)_1, dots.c, bold(a)_n)$なる1次独立なベクトルの組$bold(a)_1, dots.c, bold(a)_n$を求める．（$L(bold(a)_1, dots.c, bold(a)_n)$は$bold(a)_1, dots.c, bold(a)_n$によって生成される$R^2$の部分空間である．）

    　ベクトルの組$bold(a)_1, dots.c, bold(a)_n$は標準基底に関する$F$の表現行列である．$mat(delim: "(", F(mat(delim: "(", 1; 0; 0)), F(mat(delim: "(", 0; 1; 0)), F(mat(delim: "(", 0; 0; 1))) = mat(delim: "(", 1, 0; 0, 1) mat(delim: "(", 1, 1, 3; 2, 3, 4)$であるから，$A = mat(delim: "(", 1, 1, 3; 2, 3, 4) = (bold(a)_1 bold(a)_2 bold(a)_3) "とする" $が表現行列である．すると，$"Im" F = L(bold(a)_1, bold(a)_2, bold(a)_3)$である．

    　$"dim" ("Im" F) <= "dim" R^2 = 2$より，$"Im" F$の1次独立なベクトルの数は最大2である．$bold(a)_1 "と" bold(a)_2$は1次独立なので，$L(bold(a)_1, bold(a)_2, bold(a)_3) = L(bold(a)_1, bold(a)_2)$である．よって，$"Im" F = L(bold(a)_1, bold(a)_2)$であり，求める基底は$angle.l bold(a)_1, bold(a)_2 angle.r$．

2. $"Ker" F$の基底を１つ求める．

    　$"Ker" F$の基底を１つ求める．すなわち，$A bold(x) = bold(0) "なる" bold(x) in R^3$の１つを求める．

    　$mat(delim: "(", x_1 + x_2 + 3 x_3 ; 2 x_1 + 3 x_2 + 4 x_3) = mat(delim: "(", 0; 0)$を計算すると，$bold(x) = t mat(delim: "(", - 5; 2; 1)(t in R)$となる．$bold(x)$の代表として$mat(delim: "(", - 5; 2; 1)$をとると，求める基底は$mat(delim: "(", - 5; 2; 1)$．

</Steps>
