しかし，ほぼ独自の証明なので間違いがあったらすみません．

<Card title="集合の積">
　有理数を要素とする集合$S, T$に対して，$S T$を以下のように定義する．
$$
S T = {s t in QQ divides s in S, t in T}
$$
</Card>
<Card title="実数の積">
　任意の実数$alpha = angle.l A, A' angle.r, beta = angle.l B, B' angle.r$において，その積$alpha beta$を以下のとおりに定義する．
$$
alpha beta = angle.l R, R' angle.r \
"ただし，" \
alpha >= 0 "，" beta >= 0 "ならば，" R = R' "の補集合，" R' = A' B' \
alpha < 0 "，" beta < 0 "ならば，" R = A' B' "，" R' = R "の補集合" \
alpha > 0 "，" beta < 0 "ならば，" R = A' B "，" R' = R "の補集合" \
alpha < 0 "，" beta > 0 "ならば，" R = A B' "，" R' = R "の補集合"
$$
</Card>
　場合分けが４つもあるので，混乱しそうだ．難しく考えなくてよい．
切断箇所が正負で変わってしまうので仕方ないのである．

実数の積も有理数の積と同じように考えてよく，有理数直線上のどのポイントで切断するかは
実数を掛け算したところでよいということだ（上の書き方だとそれはわからないが）．

しかし，積も実数であることは言わないといけないし，４つの場合で構成方法が異なるので，
場合分けが発生する．

　また，$alpha >= 0 "，" beta >= 0$のとき，素直に$R = A B$としないのかという疑問があると思う．

これは$R = A B$とすると，$R'$が$QQ$における$R$の補集合と考えたとき，空集合になるから切断にならない．
つまり，$A B$が$QQ$と同値になってしまうのである．やっかいだ．
