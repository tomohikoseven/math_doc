<Card title="定理(無理数の稠密性)">
　無理数は数直線$RR$上至る所稠密に分布している．
</Card>

## 定理の主張
　[循環しない無限小数はすべて無理数](/real_number/016_irrational_number/)を表わすことを証明するとすぐにわかるらしい．私にはすぐにわからなかった．有理数が稠密であることは[定理1.4](/real_number/014_consistency/)でわかるが，無理数も稠密であることがこの定理の主張である．

## 証明の考え方
<Steps>
1. 稠密の確認

    　無理数が稠密であるとは，
 
    $$
    "無理数" alpha, beta, alpha < beta "が任意で与えられたとき，" alpha < gamma < beta "なる無理数" gamma "が無数に存在する．"
    $$
  
    ということ．

2. 切断の準備

    　$alpha, beta$は循環しない無限小数である．$alpha = k . k_1 k_2 dots.c k_n dots.c, beta = s . s_1 s_2 dots.c s_n dots.c$とおく．また，
    $$
    a_n &= k + k_1 /10 + k_2 /10^2 + dots.c + k_n /10^n \
    b_n &= s + s_1 /10 + s_2 /10^2 + dots.c + s_n /10^n
    $$
    とする． 
    $$
    C^prime = {r in QQ|forall n in NN; c_n = (a_n + b_n)/2 <= r}
    $$ 
    $C$は$C^prime$の補集合とすると，$angle.l C, C^prime angle.r$は切断である．（[循環しない無限小数は無理数を表わす](/real_number/016_irrational_number/)を参照．） 
3. $alpha < gamma < beta$であること． 

    　$gamma = angle.l C, C^prime angle.r$とするとき，$alpha < gamma < beta$であることを示す．すなわち，
    $$
    A subset.neq C, C subset.neq B
    $$ 
    であることを示す． 

    　定理1.5より，実数$alpha$に対して，$a < alpha <= a + 1/m "，" a in A, a + 1/m in C$なる有理数$a$と自然数$m$が存在する．$a + 1/m in.not A$で$a + 1/m in C$なので，$A != C$である．$C$の定義から，明らかに$A subset.neq C$なので，合わせると$A subset.neq C$となる． 

    　$gamma "と" beta$に対しても同様である．よって，$alpha < gamma < beta$である． 
 
4. $gamma$は無理数であること．

    　$C^prime$に最小の有理数がないことを示す．
    $$
    d_n = c_n + 1/10^n
    $$ 
    と$gamma$を考える．すべての自然数$n$において，$gamma < d_n$で，$d_(n + 1) < d_n$という性質から，$C^prime$に最小の有理数はない．

5. 無数に存在すること．

    　ここまでで，無理数$alpha, beta, alpha < beta$に対し，$alpha < gamma < beta$なる無理数が存在することがわかった．以下，$alpha < gamma$についても同様に議論をすれば，無理数が無数に存在することがわかる．
</Steps>
