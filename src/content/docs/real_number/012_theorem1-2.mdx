$$
\alpha < \beta，\beta < \gamma　ならば　\alpha < \gamma．
$$
</Card>

## 定理の主張
　実数は推移的な性質を持つということ．

## 証明
　$\alpha=\langle A, A' \rangle，\beta=\langle B, B' \rangle，\gamma=\langle C, C' \rangle$とする．
$\alpha < \beta，\beta < \gamma$から$A \subsetneq B，B \subsetneq C$なので，$A \subsetneq C$．
したがって，$\alpha < \gamma$．
