## 定理の主張
　実数は推移的な性質を持つということ．

## 証明
　$alpha = angle.l A, A' angle.r "，" beta = angle.l B, B' angle.r "，" gamma = angle.l C, C' angle.r$とする．
$alpha < beta "，" beta < gamma$から$A subset.neq B "，" B subset.neq C$なので，$A subset.neq C$．
したがって，$alpha < gamma$．
