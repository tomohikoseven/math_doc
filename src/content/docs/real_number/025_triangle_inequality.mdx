## 証明の考え方
　$-|alpha| -|beta| <= alpha + beta <=|alpha| +|beta|$を示せばよい．

　$alpha <=|alpha|, beta <=|beta|$なので，定理1.9より$alpha + beta <=|alpha| +|beta|$．

　$-|alpha| <= alpha, -|beta| <= beta$なので，同様に定理1.9より，$-|alpha| -|beta| <= alpha + beta$．
