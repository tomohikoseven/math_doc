$$
　|\alpha + \beta | \le |\alpha| + |\beta|．
$$
</Card>

## 証明の考え方
　$-|\alpha| - |\beta| \le \alpha + \beta \le |\alpha| + |\beta| $を示せばよい．

　$\alpha \le |\alpha|, \beta \le |\beta|$なので，定理1.9より$\alpha+\beta \le |\alpha| + |\beta|$．

　$-|\alpha| \le \alpha, -|\beta| \le \beta$なので，同様に定理1.9より，$-|\alpha|-|\beta| \le \alpha + \beta$．
