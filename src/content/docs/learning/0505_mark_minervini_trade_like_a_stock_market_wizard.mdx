# マーク・ミネルヴィニのトレード手法

　本記事は、マーク・ミネルヴィニの手法を学習した際のメモを整理したものである。
自分自身の理解を深めるための備忘録であり、読者がそのまま利用することを想定していない。


## 第12章 リスク管理 (1) ー リスクの性質 ー

:::tip[POINT]
<FlatList>
  - 株式市場で一貫して成功するために最も重要な要素は「<b>リスク管理</b>」である。
  - トレード計画を遵守し、自分の基準を満たす「<b>最高の機会</b>」だけに資金を投じよ。
</FlatList>
:::

### 1. リスクの数学性
:::note[損失と回復の法則]
  損失が大きくなるほど、元本を回復するために必要な利益率は加速度的に増大する。
  （例：10%の損は11%の利益で戻るが、50%の損を取り戻すには100%の利益が必要）
:::

<FlatList>
- <b>絶対防衛線の設定</b>：損失が10%を越えることを「<b>絶対に譲れない一線</b>」とし、平均損失は6〜7%以内に抑える。

- <b>効率的な改善</b>：利益率を上げようとするよりも、損失を小さく抑えることの方が、運用成績を劇的かつ確実に向上させる。
</FlatList>


### 2. 損切りの規律

<FlatList>
- 「<b>間違い</b>」<b>の定義</b>: 手法に関わらず、投資後に株価が下がったなら、それは「<b>間違ったトレード</b>」である。

- <b>早期撤退の重要性</b>：大きな損失を避ける唯一の方法は、損失が小さいうちに切ること。市場の審判を素直に受け入れ、エゴを捨てて早期に手仕舞う。

- <b>資金の同一視</b>：利益と元本を区別せず、得た利益も血の滲むような思いで稼いだ資金として厳格に管理する。
</FlatList>


### 3. 勝者のマインドセット

<FlatList>
- <b>必要経費としての損失</b>：損失はトレードというビジネスにおける「<b>必要経費</b>」である。冷静に損切りを行い、次のチャンスのために資金を守るのがプロの仕事。

- <b>確率への期待</b>：毎回のトレードを連続する取引の一部と捉え、長期的に「<b>勝ちを損失より大きくする</b>」期待値に賭ける。

- <b>待つことの贅沢</b>：条件が完璧に整うまで「<b>何もしない（賭けない）</b>」という選択ができるのが、株式市場の最大の利点である。
</FlatList>


### 結論：成功を分かつもの

投資家としての平凡さと偉大さを分かつのは、「<b>含み損への対処</b>」にある。
「間違うこと」自体は問題ではないが、それを認めず「間違ったままでいること」は致命的なミスになる。規律を守り、悪い持ち株を迅速に処分する者だけが市場で生き残ることができる。


## 第13章 リスク管理 (2) ー リスクの運用と管理 ー

:::tip[POINT]
<FlatList>
  - トレードの成否を分けるのは手法ではなく「<b>規律</b>」である。
  - 感情を排除し、事前に決めたリスク管理計画を冷徹に実行せよ。
</FlatList>
:::

### 1. リスク管理の基本理念

<FlatList>
- <b>リスクの定義</b>：リスクとは「<b>損失の可能性</b>」そのものである。目標はリスク回避ではなく、損失の範囲をコントロール可能な状態に置く「<b>リスク管理</b>」にある。

- <b>事前の準備</b>：プロは買う前に「<b>どこで逃げるか</b>」が決まっている。緊急対応策を事前に立て、新たな事態に応じて更新し続けることが成功の鍵。

- <b>エゴの排除</b>：市場が予想に反したとき、エゴを捨ててルールを守れるかどうかが、平凡な投資家と偉大な投資家を分かつ。
</FlatList>


### 2. 売買の戦術と規律

:::note[売りの規律]
<FlatList>
  - <b>絶対的損切り</b>：許容限度のストップを事前に設定し、達したら迷わず即座に執行する。

  - <b>利益の保護</b>：トレイリングストップを活用し、株価の上昇に合わせて手仕舞い価格を引き上げていく。

  - <b>強気での利食い</b>：買い疲れが見える急上昇局面や、相場にわずかな弱さが見えた直後など、買い手が多い「<b>楽なとき</b>」に売る。
</FlatList>
:::


### 3. 期待値の数学（2対1のルール）

<FlatList>
- <b>リスク・リワードの管理</b>：株価の上限は予測不能だが、損失額は事前に制御できる。常に平均損失を平均利益より低い水準に保つ。

- <b>ミネルヴィニの基準</b>：損切り幅は平均利益の半分程度に設定すべき。
  - 例：平均利益が15%なら、損切りは7.5%以内。

  - <b>10%の壁</b>：平均利益がどれだけ大きくても、10%を超える損失は「<b>トレードの何かが根本的に間違っている</b>」サインであり、即座に撤退する。

- <b>損益レシオ</b>：勝率が50%程度でも、損失を小さく抑え続ければ（損小利大）、数学的に資産は増え続ける。
</FlatList>


### 4. 資金管理とエントリー

<FlatList>
- <b>ナンピンの厳禁</b>：含み損のある銘柄に買い増しするのは「<b>敗者の戦略</b>」であり、破産への最短ルート。

- <b>段階的エントリー（試し玉）</b>：最初は通常より小さなポジションで始め、裏付け（含み益）が得られてからポジションを増やす。
  - 調子が良いときに最大額を投じ、悪いときに最小額になるようなリズムを作る。
</FlatList>


### 5. 逆境への対処と集中投資

:::note[負けが続く（勝率50%を下回る）ときの対応]
<FlatList>
  1. 損切り幅を通常より狭くする（例：通常の損切りが7-8%の場合、5-6%にする）。
  2. 少ない利益で早めに手仕舞う。
  3. レバレッジの使用を停止する。
  4. ポジションサイズを縮小し、リスク総額を減らす。
</FlatList>
:::

<FlatList>
- <b>集中投資</b>：分散化は並外れたパフォーマンスを阻害する。最高級の機会に絞り、<b>4〜6銘柄</b>に資金を集中させる。

- <b>忍耐</b>：絶好の機会が来るまで根気よく待つ。ストレスを感じるトレードは、基準・タイミング・サイズのいずれかに問題がある証左である。
</FlatList>
